This option restricts the baseline creation to the components specified here. By default, Hudson will create baselines on all components attached to the stream.

The components should be entered with their vob tag, separated by spaces: comp1@/vobtag comp2@/vobtag comp3@/vobtag ....

Important: if a composite component is specified here, clearcase will create a baseline on it on every build. Even if there were no changes in its depending components.
