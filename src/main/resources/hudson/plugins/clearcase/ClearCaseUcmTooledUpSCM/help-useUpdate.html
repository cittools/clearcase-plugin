If this option is checked:

  • With snapshot views, Hudson will try to use a snapshot view in the workspace of the job that has the name you provided. If the view does not exist, it will be created.
  • With dynamic views, Hudson will try to use a dynamic view that has the name you provided. If the view does not exist, it will be created.
  • In both cases, the views won't be deleted even if you try to do a "Wipe out workspace". But if you delete the job (and only in that case), the snapshot views (not the dynamic ones) generated by this job will be deleted, even if this option is checked.

If this option is unchecked:

  • With snapshot views, Hudson will try to find a snapshot view in the workspace of the job that has the name you provided. If it already exists, it will be deleted. A clean view will then be created.
  • With dynamic views, Hudson will try to find a dynamic view that has the name you provided. If it already exists, it will be deleted. A clean view will then be created.
  • When deleting the job or the workspace, the snapshot views (not the dynamic ones) generated by this job will be deleted (including workspaces auto-clean operation).
