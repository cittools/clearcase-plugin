Global clearcase configuration used for this build.

You can add/modify configurations in Hudson global setup. These configurations host the following settings:

  • cleartool executable name
  • View Storage Location
  • Time Window for Merging Event Records

If you let this setting on (default), Hudson will try to find a defined configuration matching the name of the node on which the build is taking place. If no matching configuration can be found in the global settings, Hudson will take the default one.
